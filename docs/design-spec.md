# 设计规范

## 设计原则

- **信息清晰**：双语字幕层级分明，一眼区分原文和译文
- **不遮挡**：字幕在画面底部，不盖住视频核心内容
- **阅读舒适**：字体大小、颜色、描边确保在各种背景下可读
- **轻量简洁**：GUI 界面功能明确，无冗余装饰

---

## 双语字幕设计

### 显示规格

```
┌──────────────────────────────────────────────┐
│               视频画面                          │
│                                               │
│                                               │
│  ┌─────────────────────────────────────────┐  │
│  │  This is an example sentence.           │  │  ← 原文（灰色小字，14px）
│  │  这是一个示例句子。                       │  │  ← 中文翻译（白色大字，20px，黑色描边）
│  └─────────────────────────────────────────┘  │
│                                               │
├───────────────────────────────────────────────┤
│              视频控制栏                         │
└───────────────────────────────────────────────┘
```

### ASS 字幕样式参数

| 属性 | 原文行 | 中文行 |
|------|--------|--------|
| 字体大小 | 14px | 20px |
| 字体颜色 | `#CCCCCC`（浅灰） | `#FFFFFF`（纯白） |
| 描边颜色 | `#000000` | `#000000` |
| 描边宽度 | 1px | 1.5px |
| 阴影 | 无 | 1px offset, #000000 |
| 字体 | Arial / Meiryo / Malgun Gothic | Microsoft YaHei / PingFang SC |
| 对齐 | 居中 | 居中 |
| 行间距 | 4px | — |
| 底部边距 | 视频高度的 10%（最小 60px） | — |

### SRT 降级样式

SRT 不支持独立样式，双语合并为一行：

```
00:00:01,000 --> 00:00:04,000
This is an example. | 这是一个示例。
```

## 交互规范

### 拖拽交互
1. 视频文件拖入 → 拖拽区边框高亮（虚线 → 实线 + 主色调）
2. 松开 → 显示文件信息，自动检测语言并预填
3. 不支持的文件类型拖入 → 边框变红 + 提示

### 处理流程交互
1. 点击"开始处理" → 按钮禁用，进度区展开
2. 阶段自动推进（音频提取 → 语音识别 → 翻译 → 字幕生成 → 烧录）
3. 点击"取消" → 弹窗确认 → 清理临时文件
4. 处理完成 → 进度区显示完成状态 + "打开文件"按钮

### 错误反馈
- FFmpeg 错误 → 弹窗提示 + 日志路径
- GPU 不可用 → 降级为 CPU 模式 + 警告标签
- 模型未下载 → 引导跳转下载页
- 磁盘空间不足 → 提前检查 + 预估所需空间

### 快捷键
- `Ctrl+O` — 选择视频文件
- `Ctrl+Enter` — 开始处理
- `Escape` — 取消处理
