# 项目需求文档

## 项目名称

ThinkSub — 视频中文字幕生成工具

## 核心痛点

观看网课时，大量视频（尤其是日语、英语、韩语视频）没有中文字幕，影响学习效率。

## 语言优先级

日语 > 英语 > 韩语

日语准确率是最高优先级，模型选型围绕日语优先展开。

## 目标用户

- 观看日/英/韩语网课的学习者
- 需要将外语视频转为中文字幕的用户

## 功能需求

### F0. 安装与模型管理
- 安装程序启动后，用户选择需要的语言组合（多选：日语 / 英语 / 韩语）
- 根据选择自动解析依赖模型，按需下载
- 核心运行时（不含模型）控制在 300MB 以内
- 用户可随时在设置中追加或删除某语言的模型
- 模型文件与应用程序分离开，便于增量更新

### F1. 视频导入
- 支持拖拽视频文件到应用窗口
- 支持点击"选择文件"按钮导入
- 支持格式：MP4、MKV、AVI、MOV、WMV、FLV、WebM
- 导入后显示视频基本信息（文件名、时长、分辨率）

### F2. 音频提取
- 使用 FFmpeg 从视频中无损提取音频
- 重采样为 16kHz 单声道 WAV（模型输入标准格式）

### F3. 语音识别 (ASR)
- 日语：使用 kotoba-whisper v2.0（日语句语专用蒸馏模型，准确率优于原版 Whisper）
- 英语 + 韩语：使用 Faster-Whisper large-v3（共享同一模型）
- 利用本地 NVIDIA GPU（CUDA）加速，引擎统一为 CTranslate2
- 语言自动检测（也可手动指定）
- 不依赖 PyTorch，全部推理走 CTranslate2

### F4. 翻译
- 日语 → 简体中文
- 英语 → 简体中文
- 韩语 → 简体中文
- 使用 OPUS-MT 模型，转换为 CTranslate2 格式运行（统一引擎）
- 全部本地离线，不上传任何数据

### F5. 字幕文件生成
- 输出 SRT 格式字幕文件（通用标准）
- 可选输出 ASS 格式（支持更多样式）
- **双语字幕**：同时显示原文和中文翻译（源语言在上，中文在下）
- 时间轴与视频音频对齐
- 字幕文件名与视频同名，保存到同目录或用户指定目录

### F6. 硬字幕烧录（可选）
- 将**双语字幕**直接烧录到视频画面中（原文在上，中文在下）
- 使用 FFmpeg 的 subtitles 滤镜
- 输出为 MP4 格式（H.264 编码）
- 烧录后不保留原视频（仅输出带字幕的新视频）

### F7. 处理模式
- **高质量模式（默认）**：日语 kotoba-whisper + 英/韩 large-v3，准确度优先
- **快速模式**：日语 kotoba-whisper + 英/韩 medium，速度提升 2-3 倍
- 用户可在开始处理前切换

### F8. 进度反馈
- 显示当前处理阶段（提取音频 → 语音识别 → 翻译 → 生成字幕 / 烧录）
- 显示整体进度百分比
- 显示预估剩余时间
- 支持取消正在进行的任务

## 非功能需求

### N1. 性能
- 高质量模式：10 分钟视频处理时间 < 5 分钟（含 ASR + 翻译）
- 快速模式：10 分钟视频处理时间 < 2 分钟
- GPU 显存峰值 < 4GB

### N2. 稳定性
- 支持处理 2 小时以上的长视频
- 大文件（> 4GB）不崩溃
- 异常情况友好报错，不闪退

### N3. 安全与隐私
- 所有处理完全本地进行
- 不上传视频、音频或字幕到任何服务器
- 不联网请求任何翻译或 ASR API

### N4. 磁盘占用

核心运行时（不含模型）：约 **290 MB**

| 语言选择 | 追加模型 | 总占用 |
|----------|----------|--------|
| 仅日语 | + 1.8 GB | **~2.1 GB** |
| 仅英语 | + 3.3 GB | **~3.6 GB** |
| 仅韩语 | + 3.3 GB | **~3.6 GB** |
| 日语 + 英语 | + 5.1 GB | **~5.4 GB** |
| 日语 + 韩语 | + 5.1 GB | **~5.4 GB** |
| 英语 + 韩语 | + 3.6 GB | **~3.9 GB** |
| 全部 | + 5.4 GB | **~5.7 GB** |

### N5. 兼容性
- 操作系统：Windows 10/11（v1 仅 Windows）
- 显卡：NVIDIA GTX 1060 及以上，显存 ≥ 6GB
- FFmpeg 内置便携版，无需用户自行安装

## 不做什么（v1 范围外）

- 不做中文语音听录（不做中文 → 中文字幕）
- 不做实时字幕 / 浏览器插件
- 不做批量处理（一次只处理一个视频）
- 不做 macOS / Linux 版本
- 不做视频剪辑 / 合并等编辑功能
- 不上传分享字幕
